## float 32비트 타입

부동 소수 타입, 정수를 넣어줘도 무리 x

## double 64비트 타입

fioat 과 같이 부동 소수 타입, 정수를 넣어줘도 무리가 없다

하지만 fioat 값을 넣어 줄 순 없다

## character

문자를 표현 유니코드를 사용

큰 따옴표 사용 “ ”

문자가 한 개가 아닌 여러개가 들어가게 되면 문자열 타입이다

## String

큰 따옴표 사용 “ ”

여러 문자 넣기 가능

+와 같은 연산자를 사용해 문자열을 합칠 수 있다
